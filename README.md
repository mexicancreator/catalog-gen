# catalog-gen

Repositorio para generar **catálogos digitales** desde una carpeta de **imágenes locales**: el diseño se plantea en **HTML/CSS** y la salida en **PDF** mediante Python (**WeasyPrint**). Incluye un **Cursor Skill** (`SKILL.md`) que guía al agente con divulgación progresiva (revisar imágenes y logo, acordar título y estilo, ejecutar scripts y completar el HTML), más **`scripts/image_processor.py`** (Pillow, manifiesto e imágenes normalizadas) y **`templates/base.html`** como base de maquetación.
